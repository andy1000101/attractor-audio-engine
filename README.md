# attractor-audio-engine
Turn chaos attractors or other 3d forms into live audio 
Run index.html 
Upload one of the json data sets from repository 
Can make your own json datasets in blender, create your 3d shape/curve and run point-collect.py in blender scripting tab 
This will make the json dataset that can then be uploaded to the site 

The tool uses two different methods for 'hearing shapes' 
One is with a 3 dimensionsional version of a Tonnetz Grid
The other uses the Euclidean distance from the origin  

I used chaos attractors to make shapes as they generated shapes with interesting varation and melodies

This is an early stage experiment which started with the idea of 'what do shapes sound like?' 
Transforming complex geometry into sounds to see what shapes sound 'good' or melodic
